# Estrutura-de-Dados-Árvore-binaria-e-de-busca
Trabalho de estrutura de dados sobre Árvores em C

Objetivo do trabalho:
Entender como funciona a estrutura de árvore em C.
Implementar uma estrutura basica de árvore binária.
Implementar operações como inserção, busca, remoção.
Implementar um algoritmo de persurso de pré-ordem, em-ordem e pós-ordem.
Implementar um menu interativo com as seguintes opções:
  1. Usar Árvore Binária Simples
  2. Usar Árvore Binária de Busca (ABB)
  3. Inserir elemento
  4. Buscar elemento
  5. Remover elemento
  6. Imprimir árvore
  7. Análise da árvore
  8. Percursos
  0. Sair
Implementar árvore binária de busca (ABB) com todas as operações.
